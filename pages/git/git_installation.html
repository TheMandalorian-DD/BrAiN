Téléchargement et installation sur Windows

1 Téléchargement https://git-scm.com/download/win
2 Installer où vous souhaitez !
3 Select components :
À cocher :
- Git LFS (Large File Support)
- Associate .git* configuration files with th default text editor
- Associate .sh files to be run with Bash
4 Adjusting your PATH environment
À cocher :
- Use Git and optional Unix tools from the Command Prompt * Elle permet d'utiliser quelques commandes unix en plus de git.
5 Next jusqu'a installation !
Taper git --version dans l'invite de commande pour voir si git a bien été installer.

Téléchargement et installation sur Linux
(Dans le terminal)

1 Mettre à jour des dépots présent dans le fichier de configuration
sudo apt-get update
2 Mettre à jour les packets installés sur le système
sudo apt-get upgrade
3 Installation php
sudo apt install git
Vérifiez que l'installation a réussi en tapant git --version.

Téléchargement et installation sur Mac OS
(Dans le terminal)

1 Téléchargement https://git-scm.com/download/mac
2 Ouvrir le .dmg
3 ouvrir le .pkg (clique droit -> ouvrir)
4 Appuyer sur continuer puis installer
5 Vérifiez que l'installation a réussi en tapant git --version dans le terminal.
