Les Conditions
Les symboles

$a == $b égalité rend vrai si $a est égal à $b
$a === $b égalité rend vrai si $a est égal à $b et si $a est du même type que $b
$a != $b différence rend vrai si $a est différent de $b
$a !== $b différence rend vrai si $a est différent de $b OU pas de même type
$a <= $b inférieur ou égal rend vrai si $a est inférieur ou égal à $b
$a < $b inférieur rend vrai si $a est strictement inférieur à $b
$a >= $b supérieur ou égal rend vrai si $a est supérieur ou égal à $b
$a > $b supérieur rend vrai si $a est strictement supérieur à $b

	}
	else
	{
		$majeur = false;
	}
?>

// Ternaire

<?php
	$age = 24;
	$majeur = ($age >= 18) ? true : false;
