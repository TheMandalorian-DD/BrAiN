Téléchargement et installation sur Windows

1 Téléchargez le dernier fichier zip PHP (version non thread-safe ) depuis https://windows.php.net/download/
2 Extrayez le contenu du fichier zip dans C:\php
3 Ajoutez C:\PHP7 à la variable d'environnement de chemin d'accès système Windows 10.
Clique droit sur Ce PC -> Propriétés -> Paramètres système avancés -> Variables d'environnement... -> Double clique Path (Variable système) -> Nouveau -> Copie du chemin d'accès du fichier : ex : C\Program Files\php -> Puis OK -> OK -> OK
Vérifiez que l'installation a réussi en tapant php -v dans l'invite de commande.

Téléchargement et installation sur Linux
(Dans le terminal)

1 Mettre à jour des dépots présent dans le fichier de configuration
sudo apt-get update
2 Mettre à jour les packets installés sur le système
sudo apt-get upgrade
3 Installation php
sudo apt install php7 ou php...
Vérifiez que l'installation a réussi en tapant php -v
