Commandes linux

Aides

man ls Appel de l’aide pour la commande ls
-h ou --help Demande d’aide pour une commande
ls --help Demande d’aide pour la commande ls

history Afficher l'historique des 50 dernières commandes utilisées
↑ ↓ flèche du haut Remonter dans les différentes commandes déjà tapées
⇄ tab Auto-complétion
clear ou ctrl + l Nettoie votre fenêtre de terminal en reléguant tout le texte au dessus
ctrl + s Stoppe l'affichage lorsqu'une commande très verbeuse "pollue" un peu le terminal
ctrl + q Reprendre l'affichage...
ctrl + d ou exit Déconnecte proprement une session ou un terminal
ctrl + k Supprime tout le texte se trouvant après le curseur et le sauvegarde dans le presse papier
ctrl + u Supprime tout le texte se trouvant avant le curseur et le sauvegarde dans le presse papier
ctrl + y Permet de coller le texte présent dans le presse papier
ctrl + r Permet de faire une recherche dans l'historique des commandes (tapez un bout de la commande que vous voulez rechercher)

sudo systemctl reboot Permet de redémarrer la machine
sudo systemctl poweroff Pour éteindre la machine
sudo shutdown +1 Pour éteindre l'ordinateur dans 1 min

Gestion de la session

sudo Exécuter une commande en tant que root
su - Passer root
hostname Affiche le nom de la machine
chmod Changer les droits sur un fichier ou un dossier
chmod a+r fich.htm Permission de lecture pour tous du fichier fich.htm
chown Change le propriétaire d'un fichier/dossier
usermod Modifie un utilisateur (options : -l pour changer le nom, -g pour lui assigner un groupe, -G pour lui assigner plusieurs groupes (séparés par des virgules), -a en complément de -g ou -G, ajouter des nouveaux groupes au lieu de tout redéfinir)
adduser Ajouter un utilisateur
passwd Changer le mot de passe d'un user | ex : passwd roger
deluser Supprimer un user (option --remove-home pour supprimer tous ses fichiers)

Manipulation des fichiers

ls Liste des fichiers du répertoire
ls -l Liste détaillée des fichiers du répertoire
ls i* Liste des fichiers du répertoire commençant par i
cd Déplacement dans l’arborescence des fichiers
cd / Positionnement sur le répertoire à partir de la racine
cd .. Positionnement sur le répertoire précédent
pwd Nom du répertoire courant
mkdir prog Création du répertoire prog
rmdir prog Effacement du répertoire prog
rm prog1.c Effacement du fichier prog1.c
rm -r Permet la suppression definitive des dossiers
touch fich.txt Crée un fichier vide de ce nom s’il n’existe pas, sinon change la date de dernière modif. du fichier
cp prog1.c prog2.c Copie du fichier prog1.c dans prog2.c
mv prog1.c prog2.c Renommage ou déplacement du fichier prog1.c en prog2.c
file prog.c Type du fichier prog.c
echo "Bonjour" > fich.txt Ecriture de ”Bonjour” dans le fichier fich.txt
ls > liste.txt Envoi de la liste des fichiers du répertoire dans liste.txt
ls >> liste.txt Envoi de la liste des fichiers du répertoire dans liste.txt mais c’est copié au bout de liste.txt
cat prog.c Liste du contenu du fichier prog.c
cat a.txt >> b.txt Copie du fichier a.txt au bout du fichier b.txt
grep "main" prog.c Affiche toutes les lignes du fichier prog.c contenant main
gref the *.txt Chercher le mot the dans un fichier portant l'extension .txt
gref -i the *.txt Chercher la lettre d'un mot qui commence par the dans un fichier portant l'extension .txt
vi prog.c édition du fichier prog.c
nano prog.c édition du fichier prog.c
find Chercher des fichiers et des dossiers
gzip fich.txt Compression du fichier fich.txt en fich.txt.gz
gunzip fich.txt.gz ou gzip -d fich.txt.gz Décompression du fichier fich.txt.gz en fich.txt

Gestion réseau

hostname -i Affiche l'ip de l'ordinateur
ip a Affiche les parametres de la carte réseau
ping En local : Elle permet de vérifier si le réseau fonctionne
En distant : Elle permet de connaître le temps de réponse d'un serveur distant
Ex: ping 192.168.1.1 ou ping www.google.fr

Gestionnaire de paquets et installations

apt-get update Mise à jour des paquets
apt-get upgrade php Mise à jour de php
apt install Pour installer une application/logiciel sur debian/ubuntu
zypper install Pour installer une application/logiciel sur opensuse
