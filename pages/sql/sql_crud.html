Opérateurs disponibles : WHERE

Opérateurs Effets Exemples
= Égal WHERE salaire = 2000
< Inférieur à WHERE salaire < 2000
> Supérieur à WHERE salaire > 2000
<= Inférieur ou égal à WHERE salaire > 2000
>= Supérieur ou égal à WHERE salaire <= 2000
<> Différent de WHERE salaire <> 2000
NOT NULL Non vide WHERE salaire IS NOT NUL
NULL Vide WHERE salaire IS NULL
LIKE Contient Les prenoms qui commencent par la lettre -> L : WHERE prenom LIKE "L%"
Les prenoms qui se terminent par -> ian : WHERE prenom LIKE "%ian"
Les prenoms qui contienent les lettres -> rin : WHERE prenom LIKE "%rin%"
IN Choix parmi les valeurs WHERE prenom IN("Ursule", "Jules", "toto")
> Tous sauf WHERE prenom NOT IN("Ursule", "Jules", "toto")
BETWEEN Entre WHERE BETWEEN '2000-01-01' AND '2010-12-31'

Fonctions

Fonctions Effets Exemples
COUNT Affiche le nombre de valeur d'un champ SELECT COUNT(*) AS nb FROM users WHERE id < 50;
MAX Affiche la valeur maximum d'un champ SELECT MAX(salaire) AS maxsalaire FROM users;
MIN Affiche la valeur minimum d'un champ SELECT MIN(salaire) AS minsalaire FROM users;
AVG Affiche le moyenne des données d'un champ SELECT AVG(salaire) AS salaire_moyen FROM users;
SUM Affiche la somme des données d'un champ SELECT SUM(salaire) AS somme_salaire FROM users;
